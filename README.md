# Task 7 – Support Vector Machines (SVM)

### Objective
Implement linear and non-linear SVM models for classification on the Breast Cancer dataset using Scikit-learn.

### Learnings
- Margin maximization
- Kernel trick
- Hyperparameter tuning
- Cross-validation

### Tools
- Python
- Scikit-learn
- NumPy
- Matplotlib

### Results
- Linear Kernel Accuracy: ~98%
- RBF Kernel Accuracy: ~99%
- Best Parameters (GridSearch): C=10, gamma=0.01, kernel='rbf'

### Visualization
- 2D PCA Projection with Decision Boundary
